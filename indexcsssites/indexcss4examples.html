      • In order to create and use a CSS file , follow the below instructions.

      1. Create a new text file with the chosen name of your CSS file.

      2. Save the file as "All Files" and write your title with .css at the end like so , - title.css. - Your file should now be visible. as a css file.

      3. link the CSS file to the HTML file in the HTML. - links should be at the end of, but contained inside the head element. -

      4. Write a CSS class in the CSS folder, with style selectors In order to first target, then style your HTML element. ( follow example section)

      Example in CSS file , targeting the p or paragraph element and using a style selection,

      p { } ( CSS targeting the element )

      .textClass { border : solid red 1px; } ( CSS targeting the class )

      5. Write a class to the element , in the HTML file that is linked , to the element you wish to target ( write without spaces)

      Example in HTML file ,

      < p class="textClass" > Text is Here < /p >

      6. when viewing your HTML page in a live server( or using a development live server) , you should now see your style applied to the element.

      NOTES : - in this example your styling would have applied a solid red border that is 1px wide all around the selected element.

      - Make sure your CSS is linked properly in the HTML file head section.

      - Correct Syntax means correct spelling. Correct spelling means corect code.

      7. You should be able to see your styling applied to your new web page! ;-). If not dont worry You may check on your code and guide yourself using the built in browser developer tools in your browser to better reference the exact code you are working on.
